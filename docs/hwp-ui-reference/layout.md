# 한글 프로그램 전체 레이아웃

## 개요

한글 프로그램은 Windows 표준 UI 구조를 따르며, 초보자도 쉽게 이해할 수 있는 인터페이스를 제공합니다.

## 화면 구성 요소

### 1. 상단 영역

#### 1-1. 제목 표시줄
- **파일명과 경로** 표시
  - 저장 안 된 문서: 빨간색
  - 자동 저장된 문서: 파란색
  - 저장 완료: 검은색
- **최소화 / 최대화 / 닫기** 버튼
- 마우스 우클릭 시:
  - 파일 경로 복사
  - 파일 경로 및 이름 복사
  - 파일 저장 위치 열기

#### 1-2. 메뉴 표시줄
기본 메뉴 구조 (한글 2020/2024 기준):

```
파일 | 편집 | 보기 | 입력 | 서식 | 쪽 | 보안 | 검토 | 도구
```

**주요 메뉴별 기능:**

- **파일**: 새로 만들기, 열기, 저장, 인쇄, PDF 저장
- **편집**: 잘라내기, 복사, 붙이기, 찾기, 찾아 바꾸기
- **보기**: 쪽 윤곽, 확대/축소, 눈금자, 격자
- **입력**: 표, 그림, 수식, 특수문자, 하이퍼링크
- **서식**: 글자 모양, 문단 모양, 스타일, 테두리/배경
- **쪽**: 쪽 설정, 여백, 쪽 나누기, 머리말/꼬리말
- **보안**: 문서 암호, 배포용 문서, 디지털 서명
- **검토**: 맞춤법, 변경 내용 추적, 메모
- **도구**: 환경 설정, 매크로, 글자 수 세기

#### 1-3. 빠른 실행 도구 모음
좌측 상단 위치, 자주 쓰는 기능 모음:
- 저장
- 실행 취소 / 다시 실행
- 인쇄
- 사용자 정의 가능

#### 1-4. 기능별 도구 상자 (리본 메뉴)
- **탭 방식** 메뉴바 (MS Office 리본과 유사)
- 각 메뉴 선택 시 해당 기능 그룹 표시
- **상황별 도구 상자**: 표/그림 선택 시 동적으로 나타남
- 닫기: `Ctrl+F1` (화면 공간 확보)

#### 1-5. 서식 도구 상자
- 글꼴, 크기, 색상, 스타일 등 자주 쓰는 서식 기능
- 항상 표시됨

### 2. 중앙 영역

#### 2-1. 가로 눈금자
- 문서 너비와 개체의 가로 위치 표시
- 여백, 들여쓰기, 내어쓰기 조절 도구
- 탭 설정

#### 2-2. 세로 눈금자
- 문서 높이와 개체의 세로 위치 표시
- 표/그림의 세로 크기 확인

#### 2-3. 편집 창 (문서 작업 영역)
- **기본**: A4 흰 배경
- 텍스트, 표, 이미지 편집 공간
- **쪽 단위** 표시 (인쇄 미리보기 형태)

#### 2-4. 작업 창 (우측, 상황에 따라 표시)
- 스타일
- 개체 속성
- 클립아트
- 문서마당

### 3. 하단 영역

#### 3-1. 문서 탭
- 여러 문서 동시 작업 가능
- 탭 전환: `Ctrl+Tab`
- 새 탭: `+` 버튼 또는 `Ctrl+Alt+T`

#### 3-2. 상황선 (상태 표시줄)
표시 정보:
- 전체 쪽 수
- 현재 쪽/단/줄/칸 위치
- 글자 수
- 삽입/수정 모드
- 변경 내용 추적 상태
- 표의 셀 번호

우클릭으로 표시 항목 선택 가능

#### 3-3. 확대/축소 도구
- **쪽 윤곽**: 전체 페이지 보기
- **폭 맞춤**: 페이지 너비에 맞춤
- **쪽 맞춤**: 페이지 전체가 보이도록
- **슬라이더**: 드래그로 확대/축소
- **단축키**: `Shift + Num +/-`
- **현재 배율** 클릭 → 배율 대화상자

### 4. 우측 영역

#### 4-1. 세로 이동 막대 (스크롤바)
- 문서 내용이 화면보다 클 때 세로 이동
- 드래그 또는 클릭

#### 4-2. 보기 선택 아이콘
- 쪽 윤곽
- 문단 부호 보이기/숨기기
- 조판 부호 보이기/숨기기
- 투명 선 보이기/숨기기
- 격자 설정
- 찾기 / 쪽 찾아가기 / 구역 찾아가기
- 스타일 / 조판 부호 / 책갈피 / 개체 찾아가기

#### 4-3. 쪽 이동 아이콘
- 위 화살표: 1쪽 앞으로
- 아래 화살표: 1쪽 뒤로
- 단축키: `Alt + Page Up/Down`

### 5. 기타 UI 요소

#### 5-1. 찾을 내용 (검색창)
- 상단 우측에 위치
- 단어 입력 후 `Enter`로 검색
- `Ctrl+F`로 찾기 대화상자

#### 5-2. 도움말
- 우측 상단 `?` 아이콘
- 웹 페이지로 연결
- 기능 설명 + 단축키 제공

## UI 디자인 특징

### 1. 색상 구조
- **기본 배경**: 흰색
- **선택 영역**: 검정/회색 반전
- **추적 변경**: 빨간색/파란색
- **상태 표시**: 빨간색(저장 안 됨), 파란색(자동 저장), 검은색(저장 완료)

### 2. 아이콘 디자인
- 직관적 아이콘 (MS Office와 유사)
- 한국형 기능은 고유 아이콘 사용
- 도구 설명 툴팁 제공

### 3. 반응형 UI
- 표/그림 선택 시 상황별 도구 자동 표시
- 문서 상태에 따른 동적 메뉴
