# 서식 도구

## 개요

한글 프로그램의 글자 서식 기능은 문서의 가독성과 시각적 완성도를 높이는 핵심 기능입니다.

## 글꼴 선택

### 한글/영문 글꼴 별도 설정
**한글의 독특한 기능**

- **한글 글꼴**: 한글 문자에만 적용
- **영문 글꼴**: 영문, 숫자, 기호에 적용
- **한자 글꼴**: 한자에 적용

**UI 위치**:
- 서식 도구 상자
- `서식 → 글자 모양` (Ctrl+L)

**예시**:
```
한글 글꼴: 바탕
영문 글꼴: Times New Roman
→ "한글ABC" 표시 시 한글은 바탕, ABC는 Times로 표시
```

### 글꼴 선택 UI
- 드롭다운 목록
- 최근 사용한 글꼴 상단 표시
- 글꼴 미리보기
- 즐겨찾기 글꼴 설정 가능

## 글자 크기

### 크기 설정 방법

#### 1. 프리셋 크기
- 8, 9, 10, 11, 12, 14, 16, 18, 20, 22, 24, 26, 28, 36, 48, 72pt

#### 2. 직접 입력
- 임의의 크기 입력 (1~999pt)

#### 3. 단축키
- **크게**: `Ctrl + ]`
- **작게**: `Ctrl + [`

### 크기 단위
- **포인트(pt)**: 기본 단위
- **기타**: mm, cm, 인치도 사용 가능

## 장평 (Character Width Scaling)

### 개념
글자의 가로 폭을 조절하는 기능 (한글 고유 기능)

### 설정 범위
- 50% ~ 200%
- 기본: 100%

### 사용 예시
- **90%**: 글자를 살짝 좁게 → 공간 절약
- **110%**: 글자를 살짝 넓게 → 강조 효과

**UI**: `서식 → 글자 모양 → 장평`

## 자간 (Letter Spacing)

### 개념
글자와 글자 사이의 간격 조절

### 설정 단위
- **%** 또는 **포인트(pt)**
- 음수 값 가능 (글자 겹침)

### 사용 예시
- **-5%**: 글자 간격 좁게
- **+10%**: 글자 간격 넓게

**UI**: `서식 → 글자 모양 → 자간`

## 글자 스타일

### 기본 스타일

#### 굵게 (Bold)
- **단축키**: `Ctrl + B`
- 아이콘: **B**

#### 기울임 (Italic)
- **단축키**: `Ctrl + I`
- 아이콘: _I_

#### 밑줄 (Underline)
- **단축키**: `Ctrl + U`
- **종류**:
  - 실선
  - 점선
  - 이중선
  - 물결선

#### 취소선 (Strikethrough)
- 글자 중앙에 선
- 수정 표시에 사용

### 고급 스타일

#### 그림자
- 글자에 그림자 효과
- 색상, 방향 설정 가능

#### 외곽선
- 글자 테두리 표시
- 색상, 두께 설정

#### 양각/음각
- 3D 효과
- 제목 강조용

## 글자 색상

### 글자색
- **위치**: 서식 도구 상자 `A` 아이콘
- **색상 선택**:
  - 기본 색상 팔레트
  - RGB 직접 입력
  - 최근 사용 색상

### 강조색 (Highlight)
- 글자 배경색
- 형광펜 효과
- 주요 내용 강조

## 위첨자 / 아래첨자

### 위첨자 (Superscript)
- **단축키**: `Ctrl + Shift + +`
- **용도**: 제곱, 각주 번호
- **예시**: x²

### 아래첨자 (Subscript)
- **단축키**: `Ctrl + =`
- **용도**: 화학식
- **예시**: H₂O

## 글자 효과

### 대소문자 변환
- 대문자로
- 소문자로
- 각 단어 첫 글자 대문자

### 전각/반각
- 전각: 가로 폭이 세로 높이와 같음
- 반각: 가로 폭이 세로 높이의 절반

### 한글-한자 변환
- 한글 입력 후 한자로 변환
- **단축키**: `한자` 키

## 서식 도구 상자 구성

### 레이아웃
```
[한글 글꼴▼] [영문 글꼴▼] [크기▼] [B] [I] [U] [글자색▼] [강조색▼] ...
```

### 자주 쓰는 버튼
- 굵게
- 기울임
- 밑줄
- 글자색
- 강조색
- 글꼴 크기 증가/감소

## 글자 모양 대화상자

### 접근 방법
- **메뉴**: `서식 → 글자 모양`
- **단축키**: `Ctrl + L`
- **우클릭**: 선택 영역 우클릭 → 글자 모양

### 탭 구성

#### 1. 기본 탭
- 글꼴 (한글/영문/한자)
- 크기
- 장평, 자간
- 글자 위치 (위첨자/아래첨자)

#### 2. 속성 탭
- 굵게, 기울임, 밑줄
- 취소선
- 그림자, 외곽선
- 양각, 음각

#### 3. 테두리 탭
- 글자 테두리 설정
- 색상, 선 종류

#### 4. 음영 탭
- 글자 배경색
- 그라데이션

## 스타일

### 스타일 개념
미리 정의된 서식 세트

### 기본 제공 스타일
- 본문
- 제목 1, 2, 3
- 강조
- 인용

### 스타일 적용
- **위치**: 서식 도구 상자 왼쪽
- **단축키**: `Alt + T`

### 스타일 편집
- 기존 스타일 수정
- 새 스타일 만들기
- 스타일 가져오기/내보내기

- 모든 서식 제거
- 기본 글꼴/크기로 복원
