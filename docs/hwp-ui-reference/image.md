# 이미지 / 그리기

## 개요

한글 프로그램은 이미지 삽입과 그리기 도구를 제공하여 시각적으로 풍부한 문서를 작성할 수 있습니다.

## 이미지 삽입

### 삽입 방법

#### 방법 1: 메뉴
1. `입력 → 그림 → 그림 넣기`
2. 파일 선택 대화상자
3. 이미지 파일 선택 (JPG, PNG, GIF, BMP 등)
4. "열기" 클릭

#### 방법 2: 드래그 앤 드롭
- 파일 탐색기에서 이미지 파일을 한글 문서로 드래그
- 가장 빠른 방법

#### 방법 3: 복사/붙이기
- 다른 프로그램에서 이미지 복사
- 한글에서 `Ctrl + V`

### 지원 이미지 형식
- **래스터**: JPG, PNG, GIF, BMP, TIFF
- **벡터**: SVG, EMF, WMF

## 이미지 크기 조절

### 방법 1: 핸들 드래그
- 이미지 클릭 → 8개 조절 핸들 표시
- **모서리 핸들**: 비율 유지하며 크기 조절
- **변 핸들**: 가로 또는 세로만 조절

### 방법 2: 정확한 크기 지정
1. 이미지 더블클릭 → "개체 속성"
2. "크기" 탭
3. 너비/높이 입력 (mm, cm, %, px)
4. "가로·세로 비율 유지" 체크 → 비율 유지

### 크기 옵션
- **원본 크기로**: 원본 해상도로 복원
- **쪽 너비에 맞춤**: 페이지 폭에 맞춤
- **사용자 지정**: 임의 크기

## 본문과의 배치

### 배치 유형

#### 1. 글자처럼 취급
- **특징**: 텍스트와 동일하게 동작
- 커서 위치에 삽입
- 줄 간격 영향 받음
- **용도**: 인라인 아이콘, 작은 이미지

#### 2. 어울림 (Text Wrapping)
- **특징**: 텍스트가 이미지 주변을 감쌈
- 자연스러운 레이아웃
- **배치 위치**:
  - 왼쪽 (글은 오른쪽)
  - 오른쪽 (글은 왼쪽)
  - 양쪽
  - 위
  - 아래
- **용도**: 신문형 레이아웃, 설명 문서

#### 3. 자리 차지
- **특징**: 이미지가 있는 줄에 다른 내용 불가
- 이미지만 독립적으로 위치
- **용도**: 큰 이미지, 차트

#### 4. 글 앞으로
- 이미지가 텍스트 위에 덮임
- 텍스트 가려짐
- **용도**: 워터마크, 로고 오버레이

#### 5. 글 뒤로
- 이미지가 텍스트 뒤에 위치
- 배경 이미지처럼 사용
- **용도**: 배경, 워터마크

### 배치 설정 방법
1. 이미지 더블클릭
2. "본문과의 배치" 탭
3. 배치 유형 선택
4. 본문 위치 선택 (왼쪽/오른쪽/양쪽)

### 기본 배치 설정
- `도구 → 환경 설정 → 그림`
- 기본 배치 유형 지정
- 이후 삽입되는 이미지에 적용

## 이미지 이동

### 자유 이동
- "글자처럼 취급" 해제
- 드래그하여 원하는 위치로 이동

### 정확한 위치 지정
- 개체 속성 → "위치" 탭
- X, Y 좌표 입력
- 기준점: 종이, 본문, 문단 등

### 개체 이동 안내선
- `보기 → 개체 이동 안내선`
- 이미지 이동 시 정렬 가이드 표시
- 다른 개체와 정렬 쉬워짐

## 이미지 회전 및 자르기

### 회전
- 이미지 선택 → 상단 회전 핸들 드래그
- 또는 개체 속성 → 각도 입력
- 90도 회전 버튼

### 자르기 (Crop)
1. 이미지 선택
2. 우클릭 → "그림 자르기"
3. 핸들 드래그하여 자를 영역 선택
4. Enter 또는 외부 클릭

### 대칭 이동
- 좌우 대칭
- 상하 대칭

## 이미지 효과

### 투명도
- 0% (불투명) ~ 100% (완전 투명)
- 배경 이미지에 유용

### 밝기 / 대비
- 밝기 조절: -100 ~ +100
- 대비 조절: -100 ~ +100

### 그림자
- 그림자 추가
- 색상, 방향, 거리 설정

### 테두리
- 이미지 주변 테두리
- 선 종류, 굵기, 색상

## 그리기 도구

### 도형 삽입

#### 기본 도형
- **사각형**: 일반, 둥근 모서리
- **원형**: 원, 타원
- **선**: 직선, 화살표, 곡선
- **다각형**: 삼각형, 오각형, 별

#### 도형 삽입 방법
1. `입력 → 도형`
2. 원하는 도형 선택
3. 문서에 드래그하여 그리기

### 도형 속성

#### 채우기
- 단색
- 그라데이션
- 무늬
- 그림

#### 선
- 선 종류: 실선, 점선, 파선
- 굵기: 0.1mm ~ 10mm
- 색상
- 화살표 (시작/끝)

#### 효과
- 그림자
- 반사
- 3D 효과

### 텍스트 상자

#### 삽입
- `입력 → 글상자`
- 드래그하여 크기 지정

#### 특징
- 텍스트 입력 가능
- 자유롭게 위치 이동
- 회전 가능
- 테두리/배경 설정

#### 용도
- 강조 문구
- 부가 설명
- 레이아웃 요소

### 글맵시 (WordArt)
- 꾸밈 글자
- 다양한 스타일 템플릿
- 색상, 외곽선, 그림자 효과
- **용도**: 제목, 포스터

## 개체 정렬

### 정렬 옵션
- **왼쪽 정렬**
- **가운데 정렬** (가로)
- **오른쪽 정렬**
- **위쪽 정렬**
- **중간 정렬** (세로)
- **아래쪽 정렬**

### 균등 분포
- 가로 균등 분포
- 세로 균등 분포

### 정렬 기준
- 종이
- 본문 영역
- 선택한 개체

## 개체 그룹화

### 그룹화
- 여러 개체를 하나로 묶음
- 함께 이동/크기 조절
- **방법**:
  1. 여러 개체 선택 (Shift + 클릭)
  2. 우클릭 → "개체 묶기"

### 그룹 해제
- 우클릭 → "개체 묶기 풀기"

## 개체 순서

### 앞/뒤 순서
- **맨 앞으로**: 모든 개체 위
- **맨 뒤로**: 모든 개체 아래
- **앞으로 가져오기**: 한 단계 위
- **뒤로 보내기**: 한 단계 아래

## 차트

### 차트 삽입
- `입력 → 차트`
- 차트 종류 선택: 막대, 선, 원형, 영역 등
- 데이터 입력

### 차트 편집
- 더블클릭 → 편집 모드
- 데이터 수정
- 스타일 변경

## HanDoc 구현 시사점

### Viewer 모드
- 이미지 렌더링 (JPG, PNG)
- 배치 유형별 레이아웃 계산
- 회전, 투명도 적용
- 도형 렌더링 (SVG, Canvas)

### Editor 모드
- 이미지 업로드 UI
- 크기 조절 핸들
- 배치 유형 변경 UI
- 회전/자르기 도구
- 그리기 도구 팔레트
- 개체 속성 패널

### 렌더링 최적화
- 이미지 캐싱 (Base64 또는 URL)
- 해상도별 이미지 로딩
- Canvas API로 효과 적용
- 큰 이미지 리사이징

### 데이터 구조
```typescript
interface ImageObject {
  type: 'image' | 'shape' | 'textbox';
  src: string; // 이미지 URL 또는 Base64
  position: { x: number; y: number };
  size: { width: number; height: number };
  rotation: number;
  layout: 'inline' | 'wrap' | 'square' | 'front' | 'back';
  wrapPosition: 'left' | 'right' | 'both';
  transparency: number;
  border?: Border;
  shadow?: Shadow;
}
```
