# 한컴독스 웹 에디터 분석

## 개요

한컴독스(Hancom Docs)는 한글과컴퓨터에서 제공하는 클라우드 기반 오피스 서비스입니다.
웹 브라우저에서 한글(HWP) 문서를 편집할 수 있는 온라인 에디터를 제공합니다.

## 서비스 정보

### 기본 정보
- **URL**: https://www.hancomdocs.com
- **지원 브라우저**: Chrome, Edge, Safari, Firefox
- **플랫폼**: Windows, macOS, Linux, Android, iOS
- **요금**: 무료 플랜 + 유료 구독

### 제공 서비스
- 한글 (HWP) 웹 에디터
- 한셀 (엑셀 호환)
- 한쇼 (파워포인트 호환)
- 클라우드 저장소
- 협업 기능

### 주요 UI 특징 (예상)

#### 1. 상단 메뉴바
- **파일**: 새로 만들기, 열기, 저장, 다운로드, 인쇄
- **편집**: 실행 취소, 다시 실행, 잘라내기, 복사, 붙이기
- **보기**: 확대/축소, 전체 화면
- **삽입**: 표, 이미지, 도형
- **서식**: 글꼴, 문단, 스타일

#### 2. 빠른 도구 모음
- 저장 (자동 저장)
- 실행 취소 / 다시 실행
- 글꼴 선택
- 글자 크기
- 굵게, 기울임, 밑줄
- 정렬 (왼쪽/가운데/오른쪽/양쪽)

#### 3. 편집 영역
- 쪽 단위 표시
- WYSIWYG 편집
- 실시간 미리보기

#### 4. 사이드바 (옵션)
- 파일 목록
- 공유 설정
- 버전 관리
- 댓글/메모

## 한컴독스 웹 에디터 기술 스택 (추정)

### 프론트엔드
- **프레임워크**: React 또는 Vue.js
- **에디터 엔진**: 자체 개발 또는 ProseMirror/Slate 기반
- **렌더링**: Canvas 또는 SVG
- **실시간 동기화**: WebSocket 또는 WebRTC

### 백엔드
- **서버**: Node.js 또는 Java
- **데이터베이스**: PostgreSQL, MongoDB
- **파일 저장소**: AWS S3 또는 자체 스토리지
- **협업 동기화**: OT (Operational Transformation) 또는 CRDT

## HanDoc 구현 시사점

### 웹 에디터 설계

#### 1. 렌더링 엔진
- Canvas 기반 또는 DOM 기반
- 페이지 단위 렌더링
- 가상 스크롤 (성능 최적화)

#### 2. 편집 기능
- 텍스트 입력 (IME 지원)
- 서식 적용 (리치 텍스트)
- 표, 이미지 편집
- 실행 취소/다시 실행 (히스토리 관리)

#### 3. 파일 포맷
- HWP 파싱 라이브러리
- HWPX (XML 기반)
- DOCX 변환

#### 4. 협업 기능 (선택)
- WebSocket 실시간 동기화
- OT 알고리즘
- 권한 관리

### 참고 오픈소스

#### 웹 에디터
- **ProseMirror**: 구조적 에디터 프레임워크
- **Slate.js**: React 기반 에디터
- **Quill**: 리치 텍스트 에디터
- **TinyMCE**: WYSIWYG 에디터

## 결론

한컴독스 웹 에디터는 기본적인 한글 문서 편집 기능을 웹에서 제공하며,
협업 기능에 강점이 있습니다. HanDoc은 이를 참고하되:

1. **Viewer 우선**: 완벽한 렌더링
2. **Editor 점진적 구현**: 기본 편집부터
3. **협업 옵션**: 초기에는 제외, 추후 추가
4. **성능 중시**: 빠른 로딩, 부드러운 편집
