## 🎯 프로젝트 목표

한글(HWP) 워드프로세서의 UI/UX를 체계적으로 분석하여:
- HanDoc Viewer의 읽기 모드 UI 설계
- HanDoc Editor의 편집 모드 UI 설계
- 한글 사용자들에게 친숙한 인터페이스 구현

## 📌 주요 UI 특징

### 1. 윈도우 표준 인터페이스
- Windows UI 규칙 준수
- 메뉴바 + 리본(탭형 도구상자) 구조
- 상황별 동적 도구상자

### 2. 한국형 문서 편집 최적화
- 한글/영문 글꼴 별도 설정
- 장평, 자간 세밀 조정
- 한글 맞춤법 검사

### 3. 정밀한 레이아웃 제어
- 쪽 단위 편집
- 눈금자와 격자
- 본문과의 배치 옵션 (어울림, 자리차지, 글 앞/뒤)
