# 문단 서식

## 개요

문단 서식은 텍스트의 정렬, 간격, 들여쓰기 등을 제어하여 문서의 레이아웃을 구성하는 기능입니다.

## 문단 정렬

### 정렬 방식

#### 1. 왼쪽 정렬 (Left Align)
- **단축키**: `Ctrl + Shift + L`
- 가장 일반적인 정렬
- 왼쪽 맞춤, 오른쪽 들쭉날쭉

#### 2. 가운데 정렬 (Center)
- **단축키**: `Ctrl + Shift + C`
- 제목, 표지에 사용
- 양쪽 균등 배치

#### 3. 오른쪽 정렬 (Right Align)
- **단축키**: `Ctrl + Shift + R`
- 날짜, 서명란에 사용

#### 4. 양쪽 정렬 (Justify)
- **단축키**: `Ctrl + Shift + J` 또는 `Ctrl + Shift + M`
- 양쪽 모두 맞춤
- 공백 자동 조절
- 한글 문서에서 가장 많이 사용

#### 5. 배분 정렬 (Distribute)
- 양쪽 정렬 + 마지막 줄도 양쪽 맞춤
- 표 안 텍스트에 주로 사용

### 정렬 아이콘
```
[≡]  왼쪽
[≡]  가운데
[≡]  오른쪽
[≡]  양쪽
[≡]  배분
```

## 줄 간격 (Line Spacing)

### 간격 방식

#### 1. 배수 방식
- **1.0**: 한 줄 (기본)
- **1.5**: 1.5줄
- **2.0**: 두 줄 (Double Space)

#### 2. 고정 간격
- 포인트(pt) 단위로 정확한 간격 지정
- 예: 16pt, 18pt

#### 3. 최소 간격
- 최소 간격 보장
- 글자 크기에 따라 자동 조절

### 단축키
- **줄 간격 넓게**: `Ctrl + Shift + N`
- **줄 간격 좁게**: `Ctrl + Shift + W`

### UI 위치
- 서식 도구 상자
- `서식 → 문단 모양 → 줄 간격`

## 문단 위/아래 간격

### 문단 간격
- **문단 위**: 문단 시작 전 여백
- **문단 아래**: 문단 끝 후 여백
- 단위: pt, mm, cm

### 사용 예시
```
제목 아래 여백: 12pt
본문 문단 간격: 6pt
```

## 들여쓰기

### 들여쓰기 종류

#### 1. 왼쪽 들여쓰기
- 문단 전체를 오른쪽으로 이동
- 인용구에 사용

#### 2. 첫 줄 들여쓰기
- 첫 줄만 들여쓰기
- 한글 문서 기본 스타일
- **기본값**: 10pt (약 2글자)

#### 3. 내어쓰기 (Hanging Indent)
- 첫 줄 제외하고 나머지 줄 들여쓰기
- 번호 매기기, 글머리 기호에 사용

#### 4. 오른쪽 들여쓰기
- 오른쪽 여백 조절

### 들여쓰기 설정 방법

#### 1. 눈금자 이용
- **첫 줄 들여쓰기**: 상단 삼각형 (▼) 드래그
- **왼쪽 들여쓰기**: 하단 사각형 (◼) 드래그
- **오른쪽 들여쓰기**: 오른쪽 삼각형 드래그

#### 2. 단축키
- **들여쓰기**: `Tab` (첫 줄 들여쓰기)
- **내어쓰기**: `Shift + Tab`
- **문단 들여쓰기**: `Ctrl + Shift + T`
- **문단 내어쓰기**: `Ctrl + Shift + U`

#### 3. 대화상자
- `서식 → 문단 모양`
- 정확한 수치 입력

## 번호 매기기 / 글머리 기호

### 글머리 기호 (Bullets)

#### 기본 기호
- ● (원)
- ■ (사각형)
- ◆ (마름모)
- ✓ (체크)
- → (화살표)

#### 사용자 지정
- 특수 문자 사용
- 그림 글머리 기호

**단축키**: `Ctrl + K, N` (번호 매기기/글머리 기호)

### 번호 매기기 (Numbering)

#### 번호 스타일
1. 아라비아 숫자: 1, 2, 3
2. 한글: 가, 나, 다
3. 영문 대문자: A, B, C
4. 영문 소문자: a, b, c
5. 로마 숫자: I, II, III
6. 한자: 一, 二, 三

#### 다단계 번호 매기기
```
1. 첫 번째
   1.1. 하위 항목
   1.2. 하위 항목
2. 두 번째
   2.1. 하위 항목
```

### 번호 매기기 UI
- **시작**: 번호 매기기 시작
- **계속**: 이전 번호에서 계속
- **사용자 지정**: 시작 번호 변경

## 탭 설정

### 탭 종류

#### 1. 왼쪽 탭
- 텍스트가 탭 위치에서 시작
- 기본 탭

#### 2. 가운데 탭
- 텍스트 중앙이 탭 위치

#### 3. 오른쪽 탭
- 텍스트가 탭 위치에서 끝남

#### 4. 소수점 탭
- 소수점 기준 정렬
- 숫자 정렬에 유용

### 탭 설정 방법

#### 1. 눈금자 이용
- 눈금자 클릭하여 탭 위치 추가
- 탭 마커 드래그하여 이동
- 탭 마커를 눈금자 밖으로 드래그하여 삭제

#### 2. 대화상자
- `서식 → 탭 설정`
- 정확한 위치 입력

### 탭 채움 문자
- 탭 공간을 점선, 밑줄 등으로 채움
- 목차, 가격표에 사용
- 예: `제목 ................ 3`

## 문단 모양 대화상자

### 접근 방법
- **메뉴**: `서식 → 문단 모양`
- **단축키**: `Alt + T`
- **우클릭**: 문단 우클릭 → 문단 모양

### 탭 구성

#### 1. 기본 탭
- 정렬
- 줄 간격
- 문단 위/아래 여백
- 들여쓰기

#### 2. 확장 탭
- 낱말 간격
- 한글/영문 간격
- 첫 줄 시작 문자 위치

#### 3. 테두리 탭
- 문단 테두리
- 색상, 선 종류

#### 4. 배경 탭
- 문단 배경색
- 그라데이션

## 문단 구분 기호

### 문단 끝 기호
- `Enter` 키로 생성
- 문단 부호 표시 시: ¶

### 줄 바꿈
- **단축키**: `Shift + Enter`
- 새 문단 아님, 같은 문단 내 줄 바꿈
- 문단 서식 유지

## 특수 문단 서식

### 다단 (Columns)
- 신문 형식 레이아웃
- 2단, 3단 등
- `서식 → 다단`

### 테두리/배경
- 문단에 테두리 추가
- 배경색 설정
- 강조 효과
