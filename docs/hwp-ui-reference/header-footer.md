# 머리글 / 꼬리글 / 각주

## 개요

머리글, 꼬리글, 각주는 문서의 구조와 참조 정보를 제공하는 중요한 요소입니다.

## 머리글 (Header)

### 머리글이란?
- 각 쪽 상단에 반복되는 내용
- 문서 제목, 장 제목, 날짜 등 표시
- 인쇄 시 모든 쪽에 나타남

### 머리글 편집

#### 진입 방법
1. **메뉴**: `쪽 → 머리말/꼬리말 → 머리말`
2. **더블클릭**: 페이지 상단 여백 더블클릭
3. **단축키**: `Alt + I, H`

#### 편집 모드
- 머리글 영역이 활성화됨
- 본문은 회색으로 흐려짐 (편집 불가)
- 커서가 머리글 영역에 위치

#### 편집 종료
- 본문 영역 더블클릭
- `Esc` 키
- "머리말/꼬리말 닫기" 버튼

### 머리글 내용

#### 일반적인 내용
- 문서 제목
- 장/절 제목
- 회사명/로고
- 날짜

#### 동적 내용 삽입
- **날짜**: `입력 → 날짜/시간`
- **쪽 번호**: `입력 → 쪽 번호`
- **필드**: 자동 업데이트되는 정보

### 머리글 서식
- 일반 텍스트처럼 서식 적용 가능
- 표, 이미지 삽입 가능
- 테두리, 배경 설정

### 머리글 위치 설정
- **높이**: 본문과의 간격
- **위치**: 종이 위쪽에서의 거리
- 개체 속성에서 조절

## 꼬리글 (Footer)

### 꼬리글이란?
- 각 쪽 하단에 반복되는 내용
- 쪽 번호, 저작권 표시 등

### 꼬리글 편집

#### 진입 방법
1. **메뉴**: `쪽 → 머리말/꼬리말 → 꼬리말`
2. **더블클릭**: 페이지 하단 여백 더블클릭
3. **단축키**: `Alt + I, F`

### 꼬리글 내용
- 쪽 번호 (가장 일반적)
- 작성자/작성일
- 저작권 표시
- 회사명

### 꼬리글 위치
- **높이**: 본문과의 간격
- **위치**: 종이 아래쪽에서의 거리

## 쪽 번호

### 쪽 번호 삽입

#### 삽입 방법
1. 머리글 또는 꼬리글 편집 모드 진입
2. `입력 → 쪽 번호`
3. 쪽 번호 형식 선택
4. 위치 선택 (왼쪽/가운데/오른쪽)

#### 빠른 삽입
- `입력 → 쪽 번호 → 쪽 번호 매기기`
- 대화상자에서 위치/형식 선택

### 쪽 번호 형식

#### 번호 스타일
1. **아라비아 숫자**: 1, 2, 3...
2. **로마 숫자**: I, II, III...
3. **한글**: 1쪽, 2쪽, 3쪽...
4. **영문**: Page 1, Page 2...

#### 커스텀 형식
- 예: "1 / 10" (현재 쪽 / 전체 쪽)
- 예: "페이지 1"
- 텍스트 + 쪽 번호 조합

### 쪽 번호 시작 위치
- **처음부터**: 1쪽부터 시작
- **사용자 지정**: 특정 번호부터 시작
- **새 번호로 시작**: 구역마다 다른 번호

### 쪽 번호 위치
- **상단**: 왼쪽/가운데/오른쪽
- **하단**: 왼쪽/가운데/오른쪽
- **여백**: 종이 바깥쪽

## 홀수/짝수 쪽 다르게

### 설정 방법
- `쪽 → 홀수/짝수 쪽 설정`
- 또는 구역 설정에서 지정

### 활용 예시
- **홀수 쪽**: 쪽 번호 오른쪽
- **짝수 쪽**: 쪽 번호 왼쪽
- 책 형식 문서에 유용

### 다른 내용 표시
- 홀수 쪽: 장 제목
- 짝수 쪽: 문서 제목

## 첫 쪽 다르게

### 설정
- 구역 설정에서 "첫 쪽 머리말/꼬리말 다르게" 체크

### 용도
- 표지에는 머리글/꼬리글 없음
- 본문부터 표시

## 각주 (Footnote)

### 각주란?
- 본문 하단에 표시되는 주석
- 참고 문헌, 보충 설명

### 각주 삽입

#### 삽입 방법
1. 각주를 달 위치에 커서
2. `입력 → 각주/미주 → 각주`
3. **단축키**: `Ctrl + N, N`

#### 자동 번호
- 각주 번호 자동 증가
- 위첨자로 표시
- 예: ¹, ², ³

### 각주 번호 스타일

#### 번호 형식
1. **아라비아 숫자**: 1, 2, 3
2. **로마 숫자**: i, ii, iii
3. **기호**: *, †, ‡
4. **한글**: 가, 나, 다

#### 번호 위치
- 위첨자 (기본)
- 괄호: (1), (2)
- 대괄호: [1], [2]

### 각주 위치
- **쪽 하단**: 각주가 있는 쪽 아래
- **문서 끝**: 모든 각주를 문서 끝에
- **구역 끝**: 구역마다 각주 모음

### 각주 구분선
- 본문과 각주 사이 구분선
- 선 종류, 굵기, 색상 설정

### 각주 편집
- 각주 번호 클릭 → 해당 각주로 이동
- 각주 영역에서 직접 편집

## 미주 (Endnote)

### 미주란?
- 문서 끝에 모아서 표시하는 주석
- 각주와 유사하지만 위치가 다름

### 미주 vs 각주
- **각주**: 쪽 하단 (즉시 확인)
- **미주**: 문서 끝 (모아서 확인)

### 미주 삽입
- `입력 → 각주/미주 → 미주`
- **단축키**: `Ctrl + N, E`

## 바탕쪽 (Master Page)

### 바탕쪽이란?
- 모든 쪽에 공통으로 나타나는 배경
- 로고, 워터마크 등

### 바탕쪽 편집
1. `쪽 → 바탕쪽`
2. 배경 요소 배치
3. 그림, 텍스트, 도형 삽입

### 바탕쪽 vs 머리글/꼬리글
- **바탕쪽**: 편집 불가능한 배경
- **머리글/꼬리글**: 편집 가능한 영역
