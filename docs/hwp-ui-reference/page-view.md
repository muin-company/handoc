# 페이지 보기 모드

## 개요

한글 프로그램은 문서를 다양한 방식으로 볼 수 있는 보기 모드를 제공합니다.

## 보기 모드 종류

### 1. 쪽 윤곽 (Page Layout)
**가장 일반적인 보기 방식**

- 실제 인쇄될 모습과 동일하게 표시
- 페이지 단위로 구분
- 여백, 머리말/꼬리말 모두 표시
- **위치**: 하단 우측 아이콘 또는 `보기 → 쪽 윤곽`

**특징**:
- WYSIWYG (What You See Is What You Get)
- 페이지 경계 명확하게 표시
- 기본 편집 모드

### 2. 웹 보기 (Web Layout)
- 웹 브라우저에서 볼 때와 유사한 형태
- 페이지 구분 없이 연속된 스크롤
- 웹 문서 작성 시 사용

### 3. 읽기 전용 (Read Only)
- 수정 불가, 읽기만 가능
- 문서 보호 시 사용

## 눈금자 (Ruler)

### 가로 눈금자
**위치**: 편집 영역 상단

**기능**:
1. **여백 표시**
   - 왼쪽 여백
   - 오른쪽 여백
   - 드래그로 여백 조절

2. **들여쓰기 도구**
   - 첫 줄 들여쓰기 (삼각형 ▼)
   - 왼쪽 들여쓰기 (사각형 ◼)
   - 오른쪽 들여쓰기

3. **탭 설정**
   - 탭 위치 표시
   - 클릭하여 탭 추가

4. **개체 위치**
   - 표, 그림의 가로 위치
   - 너비 확인

**단위**: mm, cm, 인치 선택 가능

### 세로 눈금자
**위치**: 편집 영역 왼쪽

**기능**:
1. **페이지 높이** 표시
2. **개체 세로 위치**
3. **표 행 높이** 확인

### 눈금자 표시/숨기기
- **메뉴**: `보기 → 눈금자`
- **효과**: 편집 영역 확대

## 확대/축소 (Zoom)

### 확대/축소 방법

#### 1. 슬라이더 (하단 우측)
- 드래그로 자유롭게 조절
- 실시간 미리보기

#### 2. 단축키
- **확대**: `Shift + Num +`
- **축소**: `Shift + Num -`

#### 3. 배율 클릭
- 현재 배율 숫자(예: 100%) 클릭
- 확대/축소 대화상자 열림

### 확대/축소 옵션

#### 쪽 맞춤
- 페이지 전체가 화면에 보이도록
- 문서 전체 구조 파악에 유용

#### 폭 맞춤
- 페이지 너비를 화면 너비에 맞춤
- 가장 많이 사용하는 옵션
- 좌우 스크롤 불필요

#### 사용자 지정
- 10% ~ 400% 범위
- 정확한 수치 입력

## 격자 (Grid)

### 격자선 표시
- **활성화**: `보기 → 격자 설정`
- 개체 정렬에 도움
- 인쇄되지 않음

**설정 옵션**:
- 격자 간격 (mm 단위)
- 색상
- 개체를 격자에 맞춤 (Snap to Grid)

## 문단 부호 / 조판 부호

### 문단 부호 보이기/숨기기
**표시 요소**:
- 문단 끝 (¶)
- 공백 (·)
- 탭 (→)

**위치**: 하단 우측 아이콘 또는 `보기` 메뉴

**용도**:
- 문서 구조 파악
- 공백/탭 확인
- 레이아웃 디버깅

### 조판 부호 보이기/숨기기
**표시 요소**:
- 구역 나눔
- 쪽 나눔
- 숨은 설명 (메모)

## 투명 선

### 투명 선 보이기
- 표의 숨겨진 테두리 표시
- 편집 시에만 보임
- 인쇄 시 나타나지 않음

## 문서 창 관리

### 여러 문서 동시 보기

#### 탭 방식 (기본)
- 상단에 탭으로 표시
- `Ctrl+Tab`으로 전환

#### 창 분할
- 가로 분할
- 세로 분할
- 계단식 배열

### 새 창에서 열기
- 같은 문서를 다른 창에서 동시 편집
- 긴 문서 편집 시 유용

## 화면 모드

### 전체 화면
- 메뉴바, 도구 모음 숨김
- 편집 영역 최대화
- `Esc`로 종료

### 크게 보기
- 상황별 도구 상자 탭 크기 증가
- 터치 환경에 유용

## 쪽 이동

### 쪽 이동 방법

#### 1. 스크롤바 이용
- 드래그로 빠른 이동
- 현재 쪽 수 표시

#### 2. 쪽 이동 아이콘
- 위 화살표: 이전 쪽
- 아래 화살표: 다음 쪽

#### 3. 단축키
- `Page Up/Down`: 화면 단위 이동
- `Alt + Page Up/Down`: 쪽 단위 이동
- `Ctrl + Home/End`: 문서 처음/끝

#### 4. 쪽 찾아가기
- `보기 → 쪽 찾아가기`
- 쪽 번호 입력하여 직접 이동

## 기타 보기 옵션

### 작업 창
- 우측에 표시되는 부가 패널
- 스타일, 개체 속성 등
- 표시/숨기기 가능

### 상태 표시줄
- 하단에 문서 정보 표시
- 우클릭으로 표시 항목 선택

## HanDoc 구현 시사점

### Viewer 모드
1. **기본 보기**: 쪽 윤곽
2. **확대/축소**: 폭 맞춤 기본
3. **쪽 이동**: 쪽 단위 스크롤 지원
4. **눈금자**: 기본 숨김 (옵션으로 표시)

### Editor 모드
1. **눈금자**: 기본 표시
2. **격자**: 옵션으로 제공
3. **문단 부호**: 토글 버튼
4. **확대/축소**: 슬라이더 + 단축키

### 성능 최적화
- 쪽 단위 렌더링 (가상 스크롤)
- 보이는 영역만 렌더링
- 확대/축소 시 리렌더링 최소화
