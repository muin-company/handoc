# @handoc/cli

CLI tool for HanDoc - inspect, extract, and convert HWP/HWPX documents.

## Installation

```bash
npm install -g @handoc/cli
```

## Usage

```bash
# Inspect HWP/HWPX file
handoc inspect document.hwpx

# Convert HWPX to DOCX
handoc convert input.hwpx output.docx

# Extract text
handoc text document.hwp
```
