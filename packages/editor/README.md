# @handoc/editor

ProseMirror-based HWPX document editor.

## Installation

```bash
npm install @handoc/editor react react-dom
```

## Usage

```tsx
import { HwpxEditor } from '@handoc/editor';

function App() {
  return <HwpxEditor document={hwpxDoc} onChange={handleChange} />;
}
```
